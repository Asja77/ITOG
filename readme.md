# Итоговый проект Data Science: определение уязвимых групп населения

## Цель
Подготовить и проанализировать открытые статистические данные, чтобы выявить потенциально уязвимые группы населения и связанные с ними социальные/криминальные риски в разрезе регионов и времени.

## Данные
1) `crimes/` — статистика Генпрокуратуры РФ по расследованным преступлениям (2016–2022):
- данные по месяцам, регионам, категориям лиц и категориям преступлений;
- файлы имеют формат `4-EGS_Razdel_%_%MMYYYY%.xls` (раздел 4/5 зависит от года);
- значения в течение года накопительные (например: февраль = январь + февраль).

2) `drug_alco/` — заболеваемость алкоголизмом и наркоманией (впервые установленный диагноз) на 100 тыс. населения, 2005–2018:
- в файле 4 листа, отдельно представлены 2017–2018 (источник отличается и названия регионов могут отличаться).

## Что сделано в проекте
- загрузка и приведение Excel-файлов к аналитическому виду (очистка, переименование столбцов, унификация регионов/категорий);
- преобразование накопительных месячных данных `crimes` в помесячные значения (расчёт разностей);
- агрегации и сравнения по регионам/годам/категориям лиц;
- подготовка витрин данных/таблиц для последующего анализа и визуализации.

## Результаты
В ноутбуке представлены очищенные датасеты и аналитические сводки, позволяющие сравнивать регионы по динамике преступности в отдельных группах (например: несовершеннолетние, ранее судимые, опьянение, иностранные граждане) и сопоставлять эти данные с медико-социальными индикаторами (алкоголизм/наркомания).

## Как запустить
Открыть `итоговая.ipynb` в Jupyter или Google Colab и выполнить ячейки последовательно.

## Структура репозитория
- `итоговая.ipynb` — основной ноутбук с обработкой и анализом
- `data/` или `crimes/`, `drug_alco/` — исходные данные по ссылке в гугл диске
- `README.md` — описание проекта
